# Growth Funnel Experiment

This project tests whether showing real-time impact insights helps developers learning growth marketing explore and understand growth funnels more effectively.

## Hypothesis

Showing insights increases engagement and helps users explore the funnel more effectively.

## The Experiment

We're running an A/B test with two variants:

- **With Insights**: Users see an Insights column that visualizes which growth levers have the biggest impact on paid users, plus personalized suggestions for what to test next
- **Without Insights**: Users see only the controls and funnel visualization, without impact data or suggestions

## What We Track

**Primary Metric**: Control changes

We measure how deeply users explore the funnel by counting how many times they toggle different growth levers. More control changes indicate greater engagement and exploration.

The simulator includes five growth stages:
- **Acquisition**: Where users come from (X, Hacker News, Paid Ads)
- **Activation**: Turning visitors into signups (CTA optimization, friction reduction)
- **Retention**: Turning signups into active users (onboarding, email reminders)
- **Revenue**: Turning active users into paying customers (free trial, pricing)
- **Referral**: Turning users into referrers (incentives, share buttons, referral loop)

## Why This Matters

For developers learning growth marketing, understanding which levers have the most impact is crucial. This experiment tests whether visualizing that impact leads to better exploration and learning compared to trial-and-error alone.
